安裝環境

Git 安裝

  1. 下載最新版Git
  2. 依照安裝步驟，下一步下一步…結束
  3. PowerShell 檢查安裝結果
    git --version

安装Node

  1. 直接下載安裝包
  2. 依照安裝步驟，下一步下一步…結束
  3. PowerShell 檢查安裝結果
    node -v

有點忘記npm 是不是要單獨安裝
npm install

安裝Visual Studio Code

  1. 上鏈接 VScode
  2. 下載完下一步連點結束
  3. 開啓 VS code 安裝插件

安装Hexo

  1. PowerShell 安裝
    npm install -g hexo-cli
  2. PowerShell 檢查安裝結果
    hexo -v

變更Hexo 主題

官網上還有很多主題可以選用哦！這邊我使用的是 A-Ayer

  2. Hexo參數調整

    /theme/ayer/_config.yml

    關於gitalk
    去 Github > Settings > Developer settings > Personal access tokens
    按下 「Generate new token」，進入新增 token 頁面
    填上 Note，捲到最下面，直接按下「Generate token」

  • hexo-generator-index-pin-top

Hexo 基本語句

  1. hexo g 生成靜態頁面
    • hexo g -d 生成文章后立即部署 Blog/public
  2. hexo s 啓動hexo 服務 本地 http://localhost:4000
    • hexo s -p 3000 設置運作端口為3000
    • hexo s -g 本地啓動前，先執行hexo g
  3. hexo clean 清理 Blog/public

上傳

  1. 首先在 GitHub 中開啓 repository

    GitHub 注冊和開專案的地方允許我引用別人的
    畢竟人家經驗豐富寫的比我好多了
    https://pengloo53.gitbooks.io/hexo/content/chapter1/8.%20%E6%89%98%E7%AE%A1%E5%88%B0Github%E6%88%96Coding.html
    依照步奏來就行

  2. 點選圖片中按鈕，複製github鏈接

3.到 Blog/public 右鍵 Git Bash here

如果repository有東西的話
在Blog 以外的地方開啓 空資料夾

1. 右鍵 Git Bash here
$ git clone {複製的鏈接}
2. clone 跑完后，剪下 .git（隱藏文件夾）
3. 貼到 Blog/public 裏面

git 熟練的可以直接在 Blog/public init 開新倉庫后，
設置remote 為剛複製的鏈接

  1. Git 三連擊
    在 Blog/public 中 Shift+滑鼠右鍵 開啓PowerShell

git的部分，是需要花時間學習的， 不論用不用，git的思想對程式設計也是有幫助的

  1. 開啓 https://{your repository}.github.io
    你的頁面就好啦~

補充

Google Ad

還沒過，過了上了再補充

Google Search

  1. 進入到 Blog 安裝sitemap 套件
    $ npm install hexo-generator-sitemap --save
  2. Blog/_config.yml 最下面貼上
  1. 然後到 Google Search Console 中注冊登入， 新增資源
  1. 點擊之後 掃描 sitemap.xml 即可錄入 google search 中啦

Google Analytics

  1. 先到 Google Analytics 首頁 登入或建立帳戶，新增一個資源，填完以後按取得追蹤編號.
  2. Blog/theme/ayer/_config.yml
    找到 google_analytics 添加申請到的追蹤編號
  3. 進入 Google Analytics 頁面點選 即時 > 總覽

再另外開個分頁瀏覽你自己的網站看看有沒有 線上使用者 跟 流量 測試一下！
