  2. 命名規則

    1. 硬性规则：
      • 變數的第一個字母一定只能是英文大小寫字母、Unicode字元或底線()，其後可以接英文大小寫字母、Unicode字元、底線()或數字
      • 對大小寫敏感
      • 關鍵字&保留字不能使用
    2. PEP 8要求：
      • 多單詞用底綫連接
      • protect 單底線(_) 開頭
      • private 雙底線(_) 開頭

  3. 變量使用

    1. #print
      a = 321
      b = 12
      print(a + b) # 333
      print(a - b) # 309
      print(a * b) # 3852
      print(a / b) # 26.75

    2. #類型
      a = 100
      b = 12.345
      c = 1 + 5j
      d = ‘hello, world’
      e = True
      print(type(a)) # <class ‘int’>
      print(type(b)) # <class ‘float’>
      print(type(c)) # <class ‘complex’>
      print(type(d)) # <class ‘str’>
      print(type(e)) # <class ‘bool’>

    3. #型別轉換 & 佔位符
      a = int(input(‘a = ‘))
      b = int(input(‘b = ‘))
      print(‘%d + %d = %d’ % (a, b, a + b))
      print(‘%d - %d = %d’ % (a, b, a - b))
      print(‘%d * %d = %d’ % (a, b, a * b))
      print(‘%d / %d = %f’ % (a, b, a / b))
      print(‘%d // %d = %d’ % (a, b, a // b))
      print(‘%d %% %d = %d’ % (a, b, a % b))
      print(‘%d ** %d = %d’ % (a, b, a ** b))
