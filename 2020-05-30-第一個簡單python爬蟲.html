一直沒有時間能夠好好學習Python，直到最近。。。
不過這文，完全是還沒好好學過就寫了，因爲老婆的公司技術不給力
給報表功能卻沒有導出的功能，所以我就越俎代庖下咯

需求

  1. 内部網站上的報表頁面上的數據拉取
  2. 轉成 execl檔案
  3. 保留一定程度的擴展（畢竟後續有問題的報表肯定不止一個）
  4. 可直接執行

過程

難點一

需要先找到報表功能的規律，包含Post請求内容的變化，及測試

這部分有資料問題就不提供了

  1. 先觀察報表欄位，幸好欄位並不變
  2. 再找到分頁參數
  3. 分別嘗試提出非前端頁面提供的數值(Ex: perPage=10000&page=1 )
  4. 將規則記錄下來，後續在程式中測試

難點二

驗證碼無法自動登入

這個真不好解決，畢竟之後老婆可能會帶去公司用，
如果登入的驗證碼都讓程式給突破了，怕給他們公司技術帶來麻煩，
所以后面讓老婆用chrome登入

難點三

取得登入的憑證

這裏應該算是運氣好，他們的站點還是用比較基本的方式驗證，所以取得Cookie 裏面的 loginUser 參數就可以了

難點四

站點要使用網絡代理驗證才能進入

這邊有個陰險的地方，花了我一點時間，從chrome裏面拿出來的cookies是cookiejar 的格式，需要經過format轉成請求的cookie

結語

最後通過PyInstaller 將pyhton編成exe就結束啦

後面的困難度就相對低了，不多説啦，上代碼

#透過pip安裝pyinstaller
pip install pyinstaller
#如果失敗可以使用以下的方法進行安裝
pip install https://github.com/pyinstaller/pyinstaller/archive/develop.tar.gz
#目前可支援Python 2.7 and 3.3—3.6
pyinstaller -F ./hello.py
